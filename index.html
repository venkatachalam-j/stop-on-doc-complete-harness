Stop on Document Complete Harness

Static GitHub Pages version for validating post-document-complete behavior in web monitors.

Recommended Flow

Run each URL twice in the same monitor type: once with Stop Test On Document Complete disabled and once with it enabled.

Compare waterfall entries after Document Complete and total duration.

Baseline

No network activity after load.

Expected: same with on/off

Late Fetch

One delayed fetch starts 500 ms after window.load.

Expected: visible when off Expected: mostly absent when on

Burst After Load

Three delayed fetches start during the legacy pause window.

Expected: several late requests when off Expected: minimal late requests when on

Long Request

A delayed fetch starts after load and downloads a larger payload to keep the request visible longer.

Expected: more visible when off Expected: reduced when on
